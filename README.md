# 内网IP存活性探测工具 

## 使用方法
```
支持A、B、C段主机探测
scan -h 192.168.0.1/24
scan -h 192.168.0.1/24 -t 100
scan -h 192.168.0.1-192.168.0.255 -t 100
```
